# Security Policy

## Supported Versions

| Version | Supported          |
|:-------:|:------------------:|
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please:

1. Open an issue in the repository with the label ```security```.
2. Add title and send email on ```danielstodulski000@gmail.com``` with:
    1. Steps to reproduce
    2. Affected versions
    3. Any relevant details.
    4. Avoid publicly disclosing the issue until it is resolved.
