# Security Policy

## Supported Versions

| Version | Supported          |
|:-------:|:------------------:|
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please:

1. Open an issue in the repository with the label security.
2. Include steps to reproduce, affected versions, and any relevant details.
3. Avoid publicly disclosing the issue until it is resolved.
